# 🚀 Star Catcher

A fun and fast-paced web game where you pilot a rocket to catch falling stars! 

### 🎮 About the Game
* **Goal**: Catch as many stars as possible to increase your score.
* **Difficulty**: The game gets faster and harder every 3 stars you catch.
* **Controls**: Use the **Arrow Keys** on your computer or the **On-screen buttons** on your mobile phone.
* **Leaderboard**: Your best scores are saved automatically to a global leaderboard via Google Sheets.
